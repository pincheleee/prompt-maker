API Troubleshooting Guide
OpenAI API Key Issues

If you're having issues with the OpenAI API:

  • Verify your API key is correct and hasn't expired
  • Check if you have sufficient credits in your OpenAI account
  • Make sure you're not hitting rate limits (free tier is limited)
  • Try regenerating your API key in the OpenAI dashboard
DeepSeek API Key Issues

If you're having issues with the DeepSeek API:

  • Verify your API key is correct
  • Check that you have sufficient usage quota remaining
  • Make sure the DeepSeek API is available (status.deepseek.com)
Network or CORS Issues

If you're experiencing network-related problems:

  • Check your internet connection
  • Try using a different browser
  • Disable any browser extensions that might interfere with API calls
  • If using a corporate network, check if API access is blocked
Alternative Options

If you can't resolve API issues:

  • Use the template-based generation method which doesn't require API keys
  • Try again later as the issue might be temporary
